Complete command-line reference for Urbanshade OS

Getting Started

The Urbanshade Terminal is a command-line interface that lets you interact with the system using text commands. Open it with Ctrl+Shift+T or from the Start Menu.

üí° Tip: Type help at any time to see a list of available commands.
urbanshade@os:~$ help
Available commands: help, clear, echo, ls, cd, cat, mkdir, touch, rm, pwd...
# Use 'help <command>' for detailed usage

File System Commands

Command Description Example
ls List directory contents ls /home
cd Change directory cd /documents
pwd Print working directory pwd
cat Display file contents cat readme.txt
mkdir Create a directory mkdir projects
touch Create an empty file touch notes.txt
rm Remove file or directory rm -r folder
cp Copy files cp file.txt backup/
mv Move/rename files mv old.txt new.txt

System Commands

Command Description Example
clear Clear the terminal screen clear
echo Print text to terminal echo "Hello!"
date Display current date/time date
whoami Display current user whoami
uptime Show system uptime uptime
version Show OS version version
reboot Restart the system reboot
shutdown Shutdown the system shutdown

Command Description Example

Command Description Example

Admin Commands

These commands require admin privileges:

Command Description

su Switch to admin user
chmod Change file permissions
kill Terminate a process
