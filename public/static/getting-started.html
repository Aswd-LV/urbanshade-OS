Welcome to Urbanshade OS

Urbanshade OS is a browser-based operating system simulation that provides a complete desktop environment experience. This guide will walk you through your first boot and help you get familiar with the system.

üí° Tip: Urbanshade OS works best on desktop browsers with JavaScript enabled. For the optimal experience, use Chrome, Firefox, Safari, or Edge.

First Boot Experience

When you first visit Urbanshade OS, you'll go through a boot sequence that mimics a real operating system:

  1. BIOS Screen
    The system displays a simulated BIOS/POST screen. Press any key or wait to continue.
  2. Boot Screen
    Watch the boot animation as system components load. This can be skipped in settings later.
  3. Installation (First Visit Only)
    A simulated installation wizard appears on your first visit. Follow the prompts to "install" the OS.
  4. OOBE - Out of Box Experience
    Configure your username, preferences, and initial settings. This is where you personalize your experience.
  5. Desktop
    Welcome to your new desktop! You'll see icons, a taskbar, and the start menu.

Essential Applications

Here are the key applications you should try first:

File Explorer

Browse the virtual file system, create folders, and manage files. Access it from the desktop or start menu.

Terminal

A command-line interface with custom commands. Try help to see available commands.

Settings

Customize your experience - themes, wallpapers, sounds, and system preferences.

Global Chat

Connect with other Urbanshade OS users in real-time (requires account).

Creating an Account

While Urbanshade OS works without an account, creating one unlocks online features:

  • Save your settings and progress across devices
  • Access global chat and messaging
  • Earn achievements and track progress
  • Participate in the Battle Pass system
  • Join leaderboards and competitions

To create an account, open the Account Settings app from the start menu.
