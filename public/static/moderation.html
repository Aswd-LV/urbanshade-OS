ü§ñ About NAVI

NAVI (Network Automated Vigilance Intelligence) is the AI-powered moderation system that monitors chat and user activity in real-time. It automatically detects and handles common violations.

NAVI Capabilities:

  • Profanity filtering
  • Spam detection
  • Link scanning
  • Rate limiting
  • Pattern recognition
  • Auto-muting
  • Escalation to mods
  • Report triage

Moderators can override NAVI decisions, adjust sensitivity, and review flagged content in the NAVI Monitor tab.

Best Practices

  • Always review context before taking action
  • Use the least severe action appropriate for the situation
  • Document reasons for all moderation actions
  • Escalate serious violations to senior moderators
  • Never discuss moderation actions publicly
  • Respect user privacy - only access what's needed
  • Follow up on reports within 24 hours

Accessing the Panel

The moderation panel is available at /moderation for authorized accounts.

Contact a senior administrator if you need access granted or have questions about your permissions.
