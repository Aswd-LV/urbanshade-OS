Quick Start: Creating an App

Here's a minimal example of creating an app for Urbanshade OS:

// src/components/apps/MyApp.tsx
import { useState } from "react";

interface MyAppProps {
  windowId: string;
}

export const MyApp = ({ windowId }: MyAppProps) => {
  return (
    <div className="h-full flex flex-col bg-background p-4">
      <h1 className="text-xl font-bold">My App</h1>
      <p className="text-muted-foreground">Hello, Urbanshade!</p>
    </div>
  );
};

Key Concepts

State Management

Urbanshade OS uses a combination of React state, localStorage for persistence, React Query for server state, and the SystemBus for event-driven communication.

Window Management

The WindowManager component handles all window operations including creation, positioning, resizing, and z-index management. Apps are rendered inside windows based on their component type.

Theming

Themes are defined using CSS custom properties in index.css. The design system uses semantic tokens (--background, --foreground, --primary, etc.) that adapt based on the active theme.
