‚ùì Frequently Asked Questions

The page won't load / shows blank
Make sure JavaScript is enabled in your browser. Urbanshade OS requires JavaScript to run. Try clearing your browser cache and refreshing. If using an ad blocker, try disabling it temporarily.
I'm stuck on the boot screen
The boot sequence can be skipped by clicking or pressing any key. If it's frozen, try refreshing the page. You can also disable the boot sequence in Settings ‚Üí System ‚Üí Skip Boot Animation.
My settings aren't saving
Settings are saved to localStorage. Make sure your browser allows localStorage. If using private/incognito mode, settings won't persist. For permanent saves, create an account for cloud sync.
Windows are disappearing or not responding
Try pressing Alt+Tab to switch windows, or Win+D to show desktop. If a window is stuck, open Task Manager (right-click taskbar) and end the unresponsive process.
I can't log into my account
Check your email and password are correct. If you forgot your password, use the reset option. Make sure you have a stable internet connection for online features.
Global chat isn't working
Global chat requires an account. If you're logged in and still having issues, check the status page for any ongoing service disruptions. Try refreshing the page.

üîÑ Reset & Recovery

‚ö†Ô∏è Warning: These steps will reset your local data. Make sure to backup important files first, or sync to cloud if you have an account.

Soft Reset

Clears temporary data while keeping settings:

  1. Open Settings ‚Üí System ‚Üí Advanced
  2. Click "Clear Cache"
  3. Refresh the page

Hard Reset

Completely resets Urbanshade OS to default state:

  1. Open Terminal
  2. Type factory-reset --confirm
  3. Or: Open browser DevTools ‚Üí Application ‚Üí Clear Storage
  4. Refresh the page

Recovery Mode

If the system won't boot normally:

  1. Hold Shift while the page loads
  2. Select "Recovery Mode" from the boot menu
  3. Use recovery tools to diagnose and fix issues

ERR_BOOT_FAILED

ERR_AUTH_FAILED

ERR_STORAGE_FULL
Local storage quota exceeded. Clear some data in Settings ‚Üí Storage.
ERR_NETWORK
Network request failed. Check internet connection and try again.
